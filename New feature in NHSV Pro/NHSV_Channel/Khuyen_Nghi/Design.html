UX Checklist

Pre-delivery theo ui-ux-pro-max rules
Touch & Accessibility
  • ✓ Sub-tab item: padding 12px → 48pt height (>44pt)
  • ✓ Filter pill: min 30pt height + 8dp gap
  • ✓ Card tap zone full row · >120pt vertical
  • ✓ PDF button: 30pt + hitSlop để đạt 44pt
  • ✓ Rating row: 44pt+ height cho tap
  • ✓ Home entry card (v1.2): mini-card carousel tap zone ≥ 60pt
  • ✓ Search input: 36pt height (+ hit area)
  • ✓ Search mini pill (Kỹ thuật): 32pt height + hitSlop để đạt 44pt
  • ✓ CTA "Xem cổ phiếu": full-width 48pt
Visual & Color
  • ✓ Lato only - không substitute
  • ✓ Market colors VN: up=#07A461 / down=#DA1004
  • ✓ Numbers: tabular-nums trên mọi price column
  • ✓ VN format: thousands "." decimal "," (e.g. 31.800)
  • ✓ % không chỉ dựa vào màu - kèm icon ▲/▼ nếu cần
  • ✓ Rating S/A/B/C/D mỗi tier có chữ + màu (no color-only)
  • ✓ Card r=12px, white bg, no shadow trên #EBECF0
  • ⚑ Test contrast: #028D96 trên white = 4.85:1 (pass)
Performance & Loading
  • ✓ Skeleton shimmer khi fetch >300ms (3 sub-tabs)
  • ✓ Rating table: client-side search debounce 250ms
  • ✓ FlatList virtualization cho rating (125+ rows)
  • ✓ Price enrichment cache 30s (khôi phục khi back)
  • ✓ PDF cache in-session (tái dùng từ NH Research)
  • ✓ Pull-to-refresh trên tất cả list screens
Content & States
  • ✓ Disclaimer luôn ở cuối screen (mọi sub-tab)
  • ✓ Empty state Kỹ thuật (v1.2: cả 14 ngày không có data) với icon đồng hồ
  • ✓ Date group header ("Hôm nay", "Hôm qua"...) phân tách rõ trong list 14 ngày
  • ✓ Search highlight matched substring (VC trong VCB)
  • ✓ Upsize ẩn nếu null (không hiển thị placeholder)
  • ✓ Vùng mua hiển thị range "X – Y" hoặc "≥ X"
  • ✓ Tap mã CK → push navigate stock detail (tái dùng)
  • ✓ "Cập nhật MM/YYYY" badge ở NHSV Rating header
  • ✓ (v1.3) % tiềm năng ≤ 0: % đỏ + badge "Đã đạt mục tiêu" (KN_09)
  • ✓ (v1.3) currentPrice null/0: ẩn %, hiển thị "Đang cập nhật giá" (KN_09)
  • ✓ (v1.3) Kỹ thuật badge động "Đã đạt mục tiêu"/"Đã chạm cắt lỗ" theo giá (KN_10)
  • ✓ (v1.3) Nhóm ngày Kỹ thuật bỏ qua Thứ 7 & Chủ nhật (20/07 → 17/07)
  • ✓ (v1.3) Báo cáo bị gỡ → "No data" tại dòng đó (KN_02)
