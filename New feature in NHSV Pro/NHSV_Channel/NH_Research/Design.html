UX Checklist

Pre-delivery theo ui-ux-pro-max rules
Touch & Accessibility
  • ✓ Card height >44pt — full row tappable
  • ✓ Category chips: min height 30pt, 8dp gap
  • ✓ PDF "Mở xem" button: 44×44pt hit area
  • ✓ Bottom nav tabs: 44pt+ tap zone
  • ✓ Header back icon: 32pt box (hitSlop 8)
  • ✓ Error retry button: 44pt min height
  • ✓ PDF zoom toolbar buttons: 36pt with hitSlop
Visual & Contrast
  • ✓ Lato font only — no font substitution
  • ✓ Title #2F4A4B on white — passes 4.5:1
  • ✓ Body #333 on white — passes 4.5:1
  • ✓ Meta #8691B3 on white — 3.4:1 (>3:1 for secondary)
  • ✓ No emoji — SVG icons from Lucide/Heroicons
  • ✓ gradient header only on authenticated screens
  • ⚑ Tabular nums cần bật cho file size & date khi dùng số
Performance & Loading
  • ✓ Skeleton shimmer thay vì spinner khi fetch >300ms
  • ✓ PDF cache in session — không tải lại lần 2
  • ✓ Infinite scroll / "Xem thêm" — lazy load
  • ✓ FlatList với virtualisation cho danh sách dài
  • ✓ PDF: trang đầu <3s, progressive render
  • ✓ Image placeholder cho thumbnail nếu có
Navigation & States
  • ✓ Back navigation: giữ scroll position khi pop
  • ✓ Deep link: nhsvpro://channel/nh-research?category=MARKET
  • ✓ Empty state: icon + title + desc (không blank)
  • ✓ Error state: icon + message + retry CTA
  • ✓ PDF Viewer: close/back rõ ràng ở header
  • ✓ Category switch: instant chip feedback + new fetch
