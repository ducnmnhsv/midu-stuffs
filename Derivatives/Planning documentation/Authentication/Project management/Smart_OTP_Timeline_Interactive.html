🔐 Smart OTP Timeline - Sequential Task Planning

Schedule tasks sequentially or in parallel with Start Week control

📝 How to Use:

  • Duration: Number of working days for the task
  • Start Week: Which week the task begins (1, 2, 3...)
  • Sequential Tasks: Task A ends Week 2 → Set Task B "Start Week" = 3
  • Parallel Tasks: Set same "Start Week" for tasks running simultaneously
  • Auto-calculation: Progress bars update automatically based on Duration + Start Week
💡 Example - Sequential Flow:
• Task A: Duration 5 days, Start Week 1 → Runs Week 1-2
• Task B: Duration 7 days, Start Week 3 → Runs Week 3-4 (starts after A)
• Task C: Duration 3 days, Start Week 5 → Runs Week 5 (starts after B)

💡 Example - Parallel Flow:
• Backend: Duration 7 days, Start Week 1 → Runs Week 1-2
• Frontend: Duration 7 days, Start Week 1 → Runs Week 1-2 (same time as Backend)
